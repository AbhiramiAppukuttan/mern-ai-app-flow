# MERN AI Flow App

## Features
- Input prompt using React Flow
- Generate AI response using OpenRouter API
- Save prompt & response to MongoDB

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js
- React Flow

## How to Run

### Backend
cd server
npm install
npm start

### Frontend
cd client
npm install
npm start

## Environment Variables

Create .env in server:

OPENROUTER_API_KEY=your_api_key
MONGO_URI=your_mongodb_uri
